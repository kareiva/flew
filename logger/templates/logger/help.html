FLEW Help - Data Entry Guide

Data Entry Examples

1. Setting Date and Time

2024-01-20                    # Sets date for subsequent QSOs
2024-01-20 14:30             # Sets both date and time
14:30                        # Sets only time
45                           # Sets minute (uses current hour)

2. Basic QSO Formats

LY2AA                        # Basic callsign entry (uses defaults)
LY2AA 599                    # Callsign with RST
14.074 LY2AA                 # Frequency and callsign
20m LY2AA                    # Band and callsign
14:30 LY2AA                  # Time and callsign
45 LY2AA                     # Minute and callsign

3. RST Variations

LY2AA 5                      # Single digit (becomes 55/559 depending on mode)
LY2AA 58                     # Two digits (becomes 58/589 depending on mode)
LY2AA 589                    # Full RST
LY2AA 599 579               # Sent and received RST

4. Contest Style

LY2AA 599 ,001              # QSO with sent serial number
LY2AA 599 ,001 .002         # QSO with sent and received serial numbers
LY2AA ,123                  # Just serial number sent
LY2AA .456                  # Just serial number received

5. Combined Formats

14.074 LY2AA 599 KO24       # Frequency, call, RST, grid
30m LY2AA 599 ,001          # Band, call, RST, serial
14:30 LY2AA 599 KO24        # Time, call, RST, grid
45 LY2AA 599 ,001 .002      # Minute, call, RST, serials

6. Setting Station Information

mycall LY1ABC               # Set your station callsign
mygrid KO24                 # Set your grid square
default mode=CW             # Set default mode
default band=20m            # Set default band
default freq=14.074         # Set default frequency

7. Multiple QSOs Example

2024-01-20 14:30
14.074 LY2AA 599 KO24
LY3BB 599
30m LY4CC 579
45 LY5DD 599 ,001
LY6EE .002

Important Notes

  • All entries are case-insensitive
  • Fields can be in any order after the callsign
  • Default values are used for any missing fields
  • Time can be in 24-hour format (HH:MM or HHMM)
  • Date formats: YYYY-MM-DD, YYYY/MM/DD, YYYY.MM.DD
  • Frequencies can be entered with or without MHz
  • Grid squares must be 4 or 6 characters
  • Serial numbers must be prefixed with comma (sent) or dot (received)

Field Descriptions

Field Format Examples
Date YYYY-MM-DD 2024-01-20, 2024/01/20
Time HH:MM or MM 14:30, 14:30:00, 45
Frequency Number[.Number] [MHz] 14.074, 14.074MHz
Band Number + m 20m, 40m, 30m
RST 1-3 digits 5, 59, 599
Grid 4-6 characters KO24, KO24ab
Serial Numbers ,NNN or .NNN ,001, .002
